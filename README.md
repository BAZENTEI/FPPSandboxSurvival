# FPPSandboxSurvival
- ジャンル: First Person Perspective Sandbox Survival Game

## 開発環境:
- Unity 2022.3.12f1  
- Visual Studio  
- plugin: DOTween
- Asset: フリー素材、フリー音源データ
- AI生成ツールの利用: なし/None 

## 操作方法
### 基本操作
- WASD: キャラの移動  
- Space: ジャンプ 
- 左/右クリック: 発射/照準
- Left Shift: キーを押しながら移動キーを押している間スタミナを消費して走る

### その他行動／キーボード・マウス
- I: インベントリを開く
- 1~8: アイテムスロット選択

### インベントリ・クラフト画面
- 左クリック: つかんでいるアイテムを全て置く

### 建築関係
- マウスホイール: スロットカーソル移動
- マウス左ボタン: ブロックを設置する
